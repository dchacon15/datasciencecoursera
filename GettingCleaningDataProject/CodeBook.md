#Code Book
This file describes the variables that are found in the the `TidiData.txt` generated by `run_analysis.R`.

##Variables

 * activity
	* Activity Type
		* Values:
			1. WALKING
			2. WALKING_UPSTAIRS
			3. WALKING_DOWNSTAIRS
			4. SITTING
			5. STANDING
			6. LAYING

		* Values:from 1 to 30
